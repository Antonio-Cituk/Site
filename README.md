### 🎈 To render & code the pug site:

#### Install the dependencies

```bash
npm i
```

#### Dev mode

```bash
npm run dev
```

#### Build the site

```bash
npm run build
```

### To commit your changes:

#### Install gitmoji-cli

```bash
npm i -g gitmoji-cli
```

#### Run -done- script
```bash
npm run done
```
